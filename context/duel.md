# Duel Context

## Temel Bilgi

- Kod anahtari: `duel`
- Oyuncu sayisi: 2
- Tip: birebir, yeniden dogmali catismali mod.

## Kurallar

- Oyuncular rakibini oldurerek skor kasar.
- Olen oyuncu kisa sure sonra yeniden dogar.
- Haritadaki normal silahlar aktiftir.
- Can paketleri aktiftir.
- Rakip ayrildiginda oda tekrar bekleme durumuna alinir.

## Kaynaklar

- Mod tanimi: `server.js` icindeki `MODES.duel`.
- Oda ayrilma davranisi: `server.js` icindeki `disconnect` akisi.
- HUD/skor: `public/game.js` icindeki skor guncelleme akisi.
