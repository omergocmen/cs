# Arena Context

## Temel Bilgi

- Kod anahtari: `arena`
- Oyuncu sayisi: 2-8
- Tip: herkes herkese serbest catisma.

## Kurallar

- Oyun basladiktan sonra yeni oyuncular katilabilir.
- Olen oyuncu yeniden dogar.
- Skor oldurme/olum uzerinden takip edilir.
- Oyuncular birbirlerinin isim etiketlerini goremez; arena icinde rakip isimleri gizlidir.
- Harita silahlari aktiftir.
- Can paketleri aktiftir.
- Airdrop kasalari aktiftir.

## Airdrop Bufflari

- Guclu silah
- Overheal
- Hizli ates
- Cift hasar

## Kaynaklar

- Mod tanimi: `server.js` icindeki `MODES.arena`.
- Airdrop kapsami: `server.js` icindeki `AIRDROP_MODES`.
- Isim etiketi gizleme: `public/game.js` icindeki `nameTagVisibleFor`.
